# Career Guidance App

1. Open the project link on GitHub.

2. Click the green *Code* button.

3. Click *Local → Download ZIP*.

4. Go to *Downloads* and locate the file:
   RTPProject.zip

5. Right-click on RTPProject.zip → Click *Extract All* → Click *Extract*.

6. Open the extracted folder:
   RTPProject

7. Inside it, find:
   FinalProject.zip
   
9. Right-click on FinalProject.zip → Click *Extract All* → Click *Extract*.

10. Open the extracted folder:
    FinalProject

11. Inside the final folder, locate:
    CareerApp.jar

12. Double-click *CareerApp.jar* to run the application.
